# 📒 Day 4 - Contact Book

## 📌 Project Overview

This project is a command-line Contact Book application built using Python. It allows users to add, view, search, and delete contacts using a menu-driven interface.

---

## 📚 Concepts Learned

- Lists vs Tuples vs Sets vs Dictionaries
- Dictionary Operations
- Functions
- While Loops
- Conditional Statements
- User Input
- CRUD Operations (Create, Read, Search, Delete)

---

## 🚀 Features

- Add Contact
- View Contacts
- Search Contact
- Delete Contact
- Exit Program

---

## 🛠️ Technologies Used

- Python 3

---

## ▶️ How to Run

```bash
python contact_book.py
```

---

## 💡 Key Learning

This project helped me understand why dictionaries are one of the most useful data structures in Python. Using key-value pairs made searching, updating, and deleting contacts efficient and intuitive while reinforcing the importance of writing modular code with functions.
