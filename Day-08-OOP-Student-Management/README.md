# 📚 Day 08 – OOP Student Management System.

## Overview

This project is part of my **60 Days Python Backend Development Challenge**.

The objective of this project was to understand the fundamentals of **Object-Oriented Programming (OOP)** by building a simple Student Management System.

---

## Features

- Add Student Details
- Display Student Information
- Update Student Marks
- Calculate Student Grade
- Display School Name (Class Attribute)
- Validate Marks (Static Method)

---

## OOP Concepts Used

- Classes
- Objects
- Constructors (`__init__`)
- `self`
- Instance Attributes
- Class Attributes
- Instance Methods
- Class Methods
- Static Methods
