# 🛠️ Day 7 – Utility Toolkit

## 📌 Project Overview

The Utility Toolkit is a command-line Python application built using a modular programming approach. Instead of writing everything in a single file, the application is divided into multiple modules, making the code easier to maintain, understand, and extend.

---

## 📚 Concepts Learned

- Python Modules
- Import Statements
- Modular Programming
- Functions
- Code Reusability
- Project Organization

---

## 🚀 Features

### 🧮 Calculator
- Addition
- Subtraction
- Multiplication
- Division

### 📏 Unit Converter
- Centimeters to Meters
- Meters to Centimeters
- Celsius to Fahrenheit
- Fahrenheit to Celsius

### 🔐 Password Generator
- Generates secure random passwords with user-defined length

### 👋 Greetings
- Good Morning
- Good Evening
- Welcome Message

---

## 🛠️ Technologies Used

- Python 3
- random module
- string module
