# 🎓 Day 3 - Student Grade System

## 📌 Project Overview

This project is a command-line Student Grade System built using Python. It accepts marks for five subjects, calculates the average score, determines the corresponding grade, and displays the result in a formatted output.

---

## 📚 Concepts Learned

- Functions
- Parameters and Arguments
- Return Values
- User Input
- Conditional Statements (`if`, `elif`, `else`)
- Basic Arithmetic Operations
- String Formatting (f-strings)

---

## 🚀 Features

- Accepts student name
- Accepts marks for five subjects
- Calculates average marks
- Assigns grades based on average
- Displays formatted results

---

## 🛠️ Technologies Used

- Python 3

---

## ▶️ How to Run

```bash
python student_grade.py
```

---

## 💡 Key Learning

This project helped me understand how to divide a program into reusable functions. I learned the difference between printing values and returning them, making the code cleaner and easier to maintain.
